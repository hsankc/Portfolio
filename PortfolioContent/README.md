# Portfolio Content Archive

This folder is Hasan Kasikci's long-term portfolio content archive.

Original files are not moved or deleted. Files copied here are organized so they can be reused later for:

- Portfolio website content
- CV updates
- LinkedIn posts
- Proof and certificate galleries
- Hackathon and project stories

## GitHub Repository

Portfolio website repository:

- https://github.com/hsankc/Portfolio

Use this repository for the website implementation and Vercel deployment flow.

## Structure

- `00_profile`: CV, profile photo candidates, bio and contact notes.
- `01_projects`: Project records and project-specific media.
- `02_hackathons`: Hackathon proof, timeline material and event assets.
- `03_certificates`: Certificates grouped by topic.
- `04_leadership`: COMU Aviation Club leadership and organization proof.
- `05_ventures`: ilion7.web, InterKey and client/business work.
- `06_media_later`: Photos and media to be added later.
- `99_raw_imports`: Raw copies of original source files.

## Public Site Rules

- Do not publish phone numbers or reference phone numbers.
- Show certificates and proof as cards or small previews, not full public downloads by default.
- Keep full-resolution originals private unless explicitly approved for public use.
- Treat COMU Aviation Club documents as a leadership proof collection, not as 23 separate noisy cards.
- Featured projects for the first home-page pass: NeuralAir, News / Truva Haber, COMU Aviation website, QuakeGrid.
