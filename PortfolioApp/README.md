# Hasan Kaşıkcı Portfolio

Modern, bilingual portfolio for `hasankasikci.com.tr`.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- lucide-react
- next-intl provider-ready i18n

## Routes

- `/` Turkish home
- `/en` English home
- `/projects` and `/en/projects`
- `/projects/[slug]` and `/en/projects/[slug]`
- `/hackathons`, `/proof`, `/writing`, `/contact`
- `/admin` V2 placeholder, noindex

## Privacy Rules

- Phone and reference phone numbers are not published.
- Full certificate PDFs are not copied into `public`.
- Leadership documents are shown as low-resolution proof previews.
- CV is kept in `PortfolioContent` as source material, not as a public download.
